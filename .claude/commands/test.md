## テスト実行

作成したテストを実行します：

```bash
# すべてのテスト実行
pytest tests/

# 特定のテストファイル
pytest tests/test_[モジュール名].py

# 特定のテストケース
pytest tests/test_[モジュール名].py::test_[テスト名]

# カバレッジ付き
pytest --cov=. tests/

# 詳細出力
pytest -v tests/
```
